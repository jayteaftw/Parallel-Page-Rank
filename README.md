# Parallel-Page-Rank

*To compile and run:*


Single-threaded:

    '''
    make singlethread
    '''

    '''
    ./singlethread [path to input file] [# of nodes] -t [# of threads]
    '''


OpenMP:

    '''
    make openmpver
    '''

    '''
    ./openmpver [path to input file] [# of nodes] -t [# of threads] [approach(1-VectorMatrixOnly, 2-MatrixMatrix)]
    '''
